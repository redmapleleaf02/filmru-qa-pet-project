# filmru - Checklist. UI and Navigation

## 1. Homepage:
- Verify correct loading of the homepage (no errors, everything displays properly);
- Check the presence of main sections:
  - Что посмотреть (What to watch);
  - Новинки онлайн (New online releases);
  - Войти (Login).
- Verify correct display of banners and promo blocks;
- Test links on the homepage (all clickable elements lead to the correct destination).

## 2. Section Navigation (Sidebar):
- Navigate to Смотреть онлайн (Watch online) -> ensure video content is displayed;
- Navigate to График выхода (Release schedule) -> verify upcoming releases are listed;
- Navigate to Коллекции (Collections) -> check that curated collections open correctly;
- Navigate to Журнал (Magazine) -> verify articles are displayed;
- Navigate to Новости (News) -> confirm current publications are available;
- Navigate to Игры (Games) -> ensure the section loads properly;
- Verify sidebar usablitiy and scrolling.

## 3. Search:
- Check the presence of the search bar (поиск по сайту - Search the site);
- Enter the title of a popular movie -> ensure results are correct;
- Enter a non-existent title -> verify the "no results found" message;
- Test filters (Все страницы (All pages), По релевантности (By relevance)).

## 4. UI/UX:
- Verify responsiveness: resize the browser window -> elements should rearrange properly;
- Check text readibility;
- Verify hover states for all buttons/links;
- Verify scrolling on long pages;
- Ensure images display correctly (no broken links).

## 5. Negative Scenarios:
- Enter special characters in the search (@@@, ###, <>);
- Test the site's response to an empty query;
- Attemt to navigate to a non-existent link (e.g., manually modify the URL).
