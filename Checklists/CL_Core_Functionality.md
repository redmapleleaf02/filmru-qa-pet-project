# filmru - Checklist. Core functionality

## 1. Editing User Profile
- Edit profile data (name, email, password);
- Upload and change profile picture;
- Verify picture and data display across all pages;
- "Delete account" button (text correctness, functionality).

## 2. Site Search
- Search by keywords (movies, series, games);
- Verify filters and sorting (by genre, year, rating);
- Search results match the query;
- Search with long/short queries and special characters.

## 3. Collections
- Create a new collection;
- Add and remove movies/series from the collection;
- Verify collection display in the profile;
- Edit collection name and description.

## 4. Reviews and Comments
- Write a review for a movie/series/game;
- Verify review display on the page;
- Ability to edit or delete a review;
- Verify review counter accuracy.

## 5. Video Content
- Play trailers and video clips;
- Verify playback quality (sound, image);
- Test "Play", "Pause", "Fullscreen" buttons;
- Verify video loading under weak internet connection.

## 6. Articles and Reviews
- Verify layout correctness (headings, paragraphs, images);
- Text readability (fonts, contrast);
- Verify links inside the article.
