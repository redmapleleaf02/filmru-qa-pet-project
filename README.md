# filmru - QA Pet Project

## Goal:
- Testing the "film.ru" website to practice manual QA and produce structured test artifacts (reports, test-plan, checklists).

## Testing Types:
- Exploratory Testing;
- UI Testing;
- Functional Testing;
- API Testing;
- Cross-Browser Testing.

## Tools:
- Postman;
- Git;
- Markdown;
- Microsoft Edge;
- Google Chrome;
- Mozilla Firefox.

## Scope:
- UI and navigation;
- Registration and login;
- Core functionality;
- API endpoints;
- Negative scenarios;
- Cross-browser compatibility.

## Structure:
- Bugs/ - bug reports and evidence (screenshots, videos);
- Checklists/ - checklists for each testing day;
- Docs/ - test plan and other documents;
- TestCases/ - formalized test cases;
- API/ - Postman collections and API logs;
- Reports/ - summary reports.

## Deliverables:
- Test Plan;
- Test Cases;
- Checklists;
- Bug Reports;
- Test Summary Report.
