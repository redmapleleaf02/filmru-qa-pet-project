# filmru – Bug Report №1

## ID: FBR-001

## Title:
- «Войти» button does not show hover effect

## Description:
- All buttons and links on the website's home page have a hover effect except the “Войти” (Login) button.

## Environment:
- OS: Windows 11;
- Browsers:
  - Microsoft Edge (version 145.0.3800.70);
  - Google Chrome (version 145.0.7632.117);
  - Mozilla Firefox (version 148.0).

## Steps:
1. Open the “film.ru” website;
2. Move the cursor over all the buttons to make sure that they have a hover effect;
3. Move the cursor over the “Войти” (Login) button.

## Expected Result:
- “Войти” (Login) button shows hover effect.

## Actual Result:
- “Войти” (Login) button does not show hover effect.

## Severity:
- Low.

## Priority:
- Minor.
