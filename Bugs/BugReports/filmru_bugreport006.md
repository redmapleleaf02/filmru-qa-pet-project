# filmru – Bug Report №6

## ID: FBR-006

## Title:
- Homepage layout – absence of responsive design when resizing browser window

## Description:
- The website does not adapt its layout when the browser window size changes. Elements remain fixed and do not rearrange themselves, which causes overlapping or hidden content. This indicates that responsive design is not implemented.

## Environment:
- OS: Windows 11;
- Browsers:
  - Microsoft Edge (version 145.0.3800.70);
  - Google Chrome (version 145.0.7632.117);
  - Mozilla Firefox (version 148.0).

## Steps:
1. Open the “film.ru” website;
2. Resize the browser window to smaller dimensions;
3. Observe the behavior of the homepage elements.

## Expected Result:
- The homepage layout is responsive: elements rearrange or resize themselves to fit the new window dimensions without overlapping or disappearing.

## Actual Result:
- The homepage layout remains static. Elements do not rearrange, some content overlaps or becomes hidden.

## Severity:
- Low.

## Priority:
- Minor.
