# filmru – Bug Report №4

## ID: FBR-004

## Title:
- Search bar – “By relevance” option does not apply other parameters (“By date”, “Alphabetical”)

## Description:
- The sorting function by parameters (“По дате” (By date), “По алфавиту” (Alphabetical)) is not available in the search bar. The basic search functionality works, but the stated parameters do not fulfill their purpose.

## Environment:

### Desktop:
- OS: Windows 11;
- Browser: Microsoft Edge (version 145.0.3800.70).

### Mobile:
- OS: Android 13;
- Browser: Google Chrome (version 145.0.7632.120).

## Steps:
1. Open the “film.ru” website;
2. Click on the search bar, type “minecraft” and press Enter;
3. Click on the “По релевантности” (By relevance) parameter;
4. Select “По дате” (By date) or “По алфавиту” (Alphabetical) parameter.

## Expected Result:
- Results are sorted by date or alphabetical. The selected parameter is displayed in the parameter box.

## Actual Result:
- Nothing has changed in the results section. The selected parameter is not displayed in the parameter box. Results are sorted by relevance.

## Severity:
- Medium.

## Priority:
- Normal.
