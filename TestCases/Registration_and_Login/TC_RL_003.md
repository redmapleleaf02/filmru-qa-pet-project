# filmru – Test Case 3

## ID: TC-RL-003

## Title: Registration of a new user with valid data

### Preconditions:
- Website is open on homepage;
- User is not registered yet.

### Steps:
1. Click on the “Войти” (Login) button;
2. Enter a valid email address in the field;
3. Check that a confirmation email with password code is received in the mailbox;
4. Enter the received password code into the form;
5. Click on the “Зарегистрироваться” (Sign up) button to complete registration.

### Expected Result:
- New account is successfully created and user is redirected to personal account page.
