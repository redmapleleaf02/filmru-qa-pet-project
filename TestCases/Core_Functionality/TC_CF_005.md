# filmru – Test Case 5

## ID: TC-CF-005

## Title: Create new collection

### Preconditions:
- User is on the personal account.

### Steps:
1. Click “Мои подборки” (My collections);
2. Click “Создать подборку” (Create collection);
3. Enter name;
4. Click “Сохранить” (Save).

### Expected Result:
- New collection appears in the “Мои подборки” section.
