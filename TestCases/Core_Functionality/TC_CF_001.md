# filmru – Test Case 1

## ID: TC-CF-001

## Title: Edit profile data

### Preconditions:
- User is logged in.

### Steps:
1. Click on the profile icon;
2. Click on the “Настройки” (Settings) button;
3. Change “Имя” (Name), “Фамилия” (Lastname), “Никнейм” (Nickname), “Пол” (Gender) and other fields;
4. Save changes. 

### Expected Result:
- Updated data is saved and displayed correctly across all pages.
